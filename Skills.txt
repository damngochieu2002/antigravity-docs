SKILLS
1. Skills là gì?
Skills là các "kỹ năng" mở rộng giúp AI Agent thực hiện tốt hơn trong các lĩnh vực chuyên biệt. Repo https://github.com/sickn33/antigravity-awesome-skills cung cấp 626+ skills sẵn sàng sử dụng.
2. Hướng dẫn cài đặt
Vị trí lưu trữ: 
* macOS/Linux: ~/.agent/skills hoặc ~/.gemini/skills
* Windows: %USERPROFILE%\.agent\skills hoặc %USERPROFILE%\.gemini\skills
Cách 1: Cài vào thư mục HOME (Khuyến nghị - không ảnh hưởng project)
npx antigravity-awesome-skills
# Skills sẽ được cài vào ~/.agent/skills (ngoài project)
# Git KHÔNG phát hiện, KHÔNG cần chỉnh .gitignore
Cách 2: Cài cho từng tool cụ thể
npx antigravity-awesome-skills --gemini   # → ~/.gemini/skills
Cách 3: Clone vào project (cần thêm vào .gitignore)
git clone https://github.com/sickn33/antigravity-awesome-skills.git .agent/skills
echo ".agent/" >> .gitignore
3. Cách sử dụng Skills
Sau khi cài, nếu muốn sử dụng skill brainstorming trong chat với agent, có thể nhắn: "Use the @brainstorming skill to help me plan a SaaS."
Hoặc muốn sử dụng skill lint-and-validate, có thể nhắn:"sử dụng skill @lint-and-validate.”
